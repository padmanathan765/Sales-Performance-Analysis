# Sales-Performance-Analysis Project Overview

Sales Performance Analysis using MS Excel and MySQL

## Tools Used

* MS Excel
* MySQL Workbench

## Dataset

The dataset contains 60 sales records with the following fields:

* OrderID
* OrderDate
* CustomerName
* Region
* Category
* ProductName
* Quantity
* UnitPrice
* TotalAmount

## Analysis Performed

* Category-wise Sales Analysis
* Region-wise Sales Analysis
* Customer-wise Sales Analysis
* Monthly Sales Trend Analysis

## Dashboard Features

* Pivot Tables
* Pivot Charts
* Sales Dashboard

## Key Skills Demonstrated

* SQL Queries
* Data Analysis
* Data Visualization
* Dashboard Creation
* Excel Reporting
